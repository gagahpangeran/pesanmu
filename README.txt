Cara install Pesanmu
1. Masukkan satu folder file ke tempat yang ingin dipakai untuk membuat web
2. Pastikan semua file terletak di dalam satu folder
	- index.php
	- main.php
	- db.php
	- comment.php
	- setting.php
	- foot.jpg
	- head.jpg
	- icon.ico
	- style.css
	- script.js
3. Edit db.php dan ganti nama host, username, password, dan nama database
4. Lalu buka setting.php dari browser sesuai alamat domain (misal: domainku.com/setting.php)
5. Jika mengalami kegagalan koneksi dengan database, coba periksa lagi
6. Jika kegagalan masih berlanjut, hubungi saya (line : gagahpangeran)
7. Jika berhasil membuat database, silakan hapus file setting.php atau pindahkan di tempat lain untuk back up
8. Silakan jalankan website pesanmu

Cara menggunakan Pesanmu
- Klik tombol Kirim Pesan dan akan muncul pop up kirim pesan
- Klik tiap pesan yang sudah terkirim untuk melihat dan mengirim komentar
- Mengirim pesan dan komentar secara anonim, siapa tau doi ngeliat pesan dari kalian :v
